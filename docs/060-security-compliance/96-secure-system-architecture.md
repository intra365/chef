# Secure System Architecture and Engineering Principles

## ISO 27001:2022 Control

**Control**: A.8.27 - Secure system architecture

## Overview

This document provides guidance for implementing and maintaining secure system architecture and engineering principles controls for the Intra365 platform in accordance with ISO 27001:2022 requirements.

## Control Objective

The objective of this control is to ensure appropriate security measures are in place to protect information assets and support business operations.

## Applicability

- **Status**: Applicable
- **Implementation Status**: In Progress
- **Owner**: CISO
- **Review Date**: Quarterly
