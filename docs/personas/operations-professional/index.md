# Operations Professionals

Operations Professionals are responsible for keeping production systems running smoothly, managing infrastructure, responding to issues, and ensuring service availability. They focus on stability, uptime, and operational excellence.

## Characteristics

### Primary Focus
- **System Stability**: Maintaining uptime and reliability
- **Issue Resolution**: Quickly diagnosing and fixing problems
- **Maintenance**: Regular system updates and health checks
- **Support**: Responding to user issues and requests

### Key Responsibilities
- Monitor system health and performance
- Respond to alerts and incidents
- Perform system maintenance and updates
- Manage backups and disaster recovery
- Handle user support requests
- Document operational procedures

### Information Needs
- **Runbooks** with step-by-step procedures
- **Troubleshooting guides** for common issues
- **System diagrams** showing dependencies
- **Alert documentation** with response procedures
- **Maintenance schedules** and procedures
- **Escalation paths** for complex issues

### Communication Style
- Prefers clear, actionable procedures
- Values quick reference guides
- Appreciates decision trees for troubleshooting
- Responds well to checklists and runbooks
